# Changelog

All notable changes to this project will be documented in this file.

## [1.1.0] - 2026-02-12

### Added
- **Search Bar**: Quickly filter items in the "Explore" (Items) view.
- **Remove from Bag**: Users can now remove items from their personal bag.
- **Explore View**: Renamed "Directory" to "Explore" and consolidated "Popularity" sorting into this view.
- **Smart Sorting**: "Explore" sorts by popularity, "My Bag" sorts by recently added.

### Changed
- Simplified navigation: Removed navigation bar for guests.
- Consolidated "Popularity" and "Categories" views into the main "Explore" and "My Bag" tabs.
